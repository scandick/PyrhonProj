# Анализ данных: психология, продуктивность, цифровое поведение

Проект учебного анализа данных: сбор и исследование структурированной информации с открытых HTML-страниц без использования API и JavaScript.

---

## Цель проекта

Собрать собственный датасет (1000+ строк, 8+ признаков) и провести разведывательный анализ с визуализацией и формулировкой гипотез по одной из следующих тем:

- психологические тесты и методики;
- инструменты продуктивности;
- цифровая зависимость и поведение в сети;
- классификация психических расстройств (МКБ-10).

---

## Стек

- `requests`, `BeautifulSoup` — HTML-парсинг
- `pandas` — структура и обработка данных
- `matplotlib`, `seaborn` — визуализация
- `scikit-learn` — кластеризация (опционально)

## Темы и источники

| Тематика | Источник | Объект (единица данных) | Признаки |
|----------|----------|--------------------------|----------|
| Психологические тесты | [testometrika.com](https://testometrika.com/tests/) | Один тест | Название, категория, автор, Pro/Free, описание, кол-во вопросов |
| Методики диагностики | [testoteka.narod.ru](http://testoteka.narod.ru/prof/0.html) | Методика | Название, автор, цель, структура, применяемость |
| Инструменты продуктивности | [toolfinder.co/tools](https://toolfinder.co/tools) | Продукт | Название, платформа, описание, категория, платность, оценка |
| Программы и аналоги | [alternativeto.net](https://alternativeto.net/category/productivity/) | Приложение | Название, описание, лайки, лицензия, ОС, тип |
| Тесты на зависимость | [netaddiction.com](https://netaddiction.com/self-tests/) | Диагностический опросник | Название, тематика, автор, целевая группа, валидность |
| Психические расстройства | [psychiatriya.ru/mkb-10](https://psychiatriya.ru/mkb-10/) | Один диагноз (МКБ-10) | Код, название, описание, симптомы, перекрёстные ссылки |

## Признаки для анализа

- Категориальные: тип методики, группа расстройства, платность, целевая аудитория  
- Числовые: длина описания, число симптомов, количество лайков  
- Текстовые: ключевые симптомы, краткие описания, классификационные признаки
