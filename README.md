## Цель проекта

Собрать собственный датасет (1000+ строк, 8+ признаков) и провести разведывательный анализ с визуализацией и формулировкой гипотез.
- Строк ≥1000 (для 4 чел.) - число фильмов
- Признаков ≥8–10
  - числовые (рейтинг, бюджет, сборы, длительность)
  - категориальные (жанры, страна производства)
  - текстовые (описание, теги)
---

- `requests`, `BeautifulSoup` — HTML-парсинг

## Признаки для анализа
Простой пример - редактируем в процессе обсуждения
| Признак | Описание |
|---------|-----------|
| title | Название фильма |
| year | Год выхода |
| genres | Список жанров |
| duration_min | Длительность фильма в минутах |
| rating_imdb | Рейтинг фильма на IMDb |
| votes_imdb | Количество голосов пользователей IMDb |
| budget | Бюджет фильма |
| box_office_worldwide | Мировые кассовые сборы |

## Примеры гипотез
## TO DO - совместными усилиями сформируем наиболее интересные гипотезы под нужды проекта

Пара примеров:
- Анализ влияния бюджета на сборы: Построить график зависимости сборов от бюджета, выявить выбросы и фильмы-«выстрелы».
- Жанровый анализ: Как жанр влияет на рейтинг? Какой жанр чаще получает награды?
- Распределение по странам и годам: Какие страны чаще производят кассовые или рейтинговые фильмы? Как менялись жанры по десятилетиям?
- Воздействие режиссера: Кто из режиссеров чаще всего делает успешные фильмы по сборам/рейтингам?
